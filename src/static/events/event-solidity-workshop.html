Blockchain McGill: Blockchain Solidity Workshop

Solidity is a high-level language interfacing with the biggest blockchain network in the world. Come join Blockchain at McGill to learn how to write smart contract code and best practices in Solidity. We will be having a web integrated IDE so bring your laptops and an internet connection ready to go. You can find how to integrate smart contracts into your UI and front end through a sample project you can find on our GitHub: https://github.com/BlockchainAtMcGill/bootcamp-shop. See you all there!
